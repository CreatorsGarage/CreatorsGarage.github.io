Recent Projects

Arduino Line Follower and edge

An RC car using IR sensors and intelligent programming which has features like following a line of particular colour. It will have the ability to avoid obstacle in its path. Further through Arduino programming, the feature of edge avoidance will also be integrated.

Gesture Robotic arm

Arduino/Raspberry pi controlled Robotic arm which will be remotely controlled through gestures. First prototype will be by developing a smart glove. The final prototype will include the integration of ML and Deep Learning so that the Robotic arm can be controlled by mere gestures without using gloves.

Website Development Project

Coding is Love

web development is about creating applications which run in a web browser. Some of those applications have most of their logic living on a web server which renders HTML, CSS, and JavaScript to create an application. Other applications only utilize the server to create their initial state, download the logic to run the application, and then use the server only to retrieve and store data. Regardless, though, of how web development is done, the basic technologies are the same: HTML, JavaScript, CSS—and a large amount of patience. Web developers today utilize about every major programming language to create web applications.
